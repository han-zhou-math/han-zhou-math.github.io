# Han Zhou Personal Website

This repository hosts Han Zhou's academic website.

## Setup
1. Go to [GitHub](https://github.com) and create a new repository named `han-zhou.github.io`.
2. Upload all files from this folder.
3. Enable GitHub Pages (Settings → Pages → Source: `main` branch → `/ (root)`).
4. Your website will be live at: https://han-zhou-math.github.io

## Local Preview (optional)
If you want to preview locally, install Jekyll and run:
```bash
bundle exec jekyll serve
```
